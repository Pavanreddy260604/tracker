\begin{abstract}
Generative AI services are currently fragmented and expensive, requiring multiple subscriptions. AI Express addresses this by providing a unified SaaS platform integrating text, image, and coding tools. Key features include automated code review, content creation tools, and watermark removal. Built with React.js, Node.js, and PostgreSQL using a microservice architecture, the platform implements a "pay-as-you-go" credit system making premium AI tools accessible to students and developers.
\end{abstract}

\begin{IEEEkeywords}
Artificial Intelligence, Multi-Modal AI, SaaS, Microservices, Code Analysis, Image Inpainting.
\end{IEEEkeywords}

\section{Introduction}
Generative AI has revolutionized digital productivity, but current solutions are fragmented and expensive. Users need multiple subscriptions for text, image, and coding tools. AI Express addresses this by providing a unified SaaS platform integrating cutting-edge AI models into a single interface.

The platform offers: (1) \textbf{Code Reviewer}---automated GitHub repository analysis; (2) \textbf{Data Analyst}---CSV/Excel processing with natural language queries; (3) \textbf{Content Tools}---script generation and grammar checking; (4) \textbf{Watermark Remover}---deep learning-based image inpainting; (5) \textbf{Utilities}---web scraper and resume builder.

Built on React.js, Node.js, and PostgreSQL with a microservice architecture, AI Express implements a "pay-as-you-go" credit system, democratizing access to premium AI tools for students and developers.

\section{Related Work}
\subsection{LLM-Based Code Review}
LLMs have significantly improved code review processes. Haratian et al.\cite{haratian2025code} demonstrated practical AI-assisted code review in industrial settings. However, existing solutions like GitHub Copilot require separate licensing, disrupting unified workflows.

\subsection{Autonomous Data Analytics}
Recent frameworks use LLM agents for end-to-end data tasks including SQL generation and visualization\cite{ieee2025dataanalyst}. While accurate, these tools remain isolated from programming utilities.

\subsection{Image Inpainting}
Deep learning techniques achieve high PSNR in watermark removal\cite{xu2023watermark}. However, such tools are typically limited to desktop applications rather than integrated web platforms.

\subsection{Research Gap}
Although individual AI models achieve high performance, they are deployed in siloed environments. AI Express addresses this gap by orchestrating heterogeneous AI services through a scalable microservices architecture with an equitable credit-based economy.

\section{System Architecture}
AI Express employs a microservices architecture (Fig. \ref{fig:architecture}) with three processing nodes:

\textbf{Client Layer:} React.js SPA handling input validation and API communication.

\textbf{Microservices:} (1) \textit{Linguistic \& Code Node}---LLM integration for NLP and static code analysis; (2) \textit{Vision Node}---deep learning inpainting for watermark removal; (3) \textit{Data Node}---structured data parsing and SQL generation.

\textbf{Credit Economy:} PostgreSQL manages user data and real-time token deduction based on computational cost.

\section{Module Implementation}
\textbf{Code Review:} GitHub API fetches repository source code; LLM agents analyze for anti-patterns, security risks, and optimization opportunities across multiple languages.

\textbf{Data Analyst:} Accepts CSV/TSV/Excel with natural language queries. Backend LLM translates queries to Pandas/SQL, executing securely to return statistics and visualizations.

\textbf{Content Tools:} Script Generator creates structured content from brief descriptions. Grammar Checker uses LLM-based error detection for syntactic correction and tone refinement.

\textbf{Watermark Removal:} Deep learning inpainting network with dynamic mask generation identifies watermark pixels, then reconstructs background using spatial frequency prediction.

\textbf{Utilities:} Resume Builder creates ATS-optimized CVs via skill-job description mapping. Web Scraper extracts structured data from URLs using intelligent parsing.

\subsection{Evaluation}
Code analysis successfully identified syntactical errors and refactoring opportunities. Data analyst achieved high success rates for well-formatted tables. Watermark removal showed high SSIM scores with minor artifact blurring on complex textures.

\subsection{Credit Economy}
PostgreSQL ACID properties were verified under concurrent load testing. Token deductions matched computation costs, confirming the pay-as-you-go model's viability versus subscription pricing.

\section{Conclusion}
AI Express successfully integrates LLM-based code review, autonomous data analytics, and computer vision into a unified SaaS platform. The microservice architecture with pay-as-you-go credit economy democratizes premium AI tools, providing an affordable alternative to fragmented subscription services.

\subsection{Future Work}
Planned enhancements include: (1) WebSocket-based collaborative workspaces; (2) Edge AI for client-side grammar checking; (3) Audio processing modules for TTS and podcast generation.
